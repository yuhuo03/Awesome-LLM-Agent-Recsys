# Contributing

Thanks for contributing!

## What to add

- Papers / preprints / theses
- Surveys / tutorials
- Benchmarks, datasets, simulators
- Open-source implementations

## Format

Please add entries in this format:

- Paper Title [YYYY-venue/arXiv] [[paper](URL)] [[code](URL)]

Optional:

- [[project](URL)] [[blog](URL)] [[demo](URL)]

## Quality guidelines

- Prefer primary sources (paper / official repo).
- If the work is not directly about **LLM Agents + Recommendation**, put it in the closest section (or propose a new one).
- Keep titles and venue/year accurate.

## Alphabetical vs chronological

- Within each section, we generally keep **reverse chronological** order.
